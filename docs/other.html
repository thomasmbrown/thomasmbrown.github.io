Better Premier League table

This presents a league table providing a more accurate representation of a teams performance early in the season. It gives an better indication of a relative result. For example drawing to Liverpool has more value than beating Burnley. It does all this without any input on what teams are deemed `good'.

Hovering over a team will indicate which teams the algorithm deems it better than (in green) and which they are doing worse than (in red). It does this by comparing them only on the games they have both played. The table is constructed by rewarding teams that have `beaten' more teams based on this metric.

As things stands this table does not handle teams that are drawing well. This will be added soon.
